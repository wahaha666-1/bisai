# 🤖 AgentForge - 企业级AI Agent工作流编排平台

## 📋 项目概述

**AgentForge（智链编排平台）** 是一个企业级的AI Agent工作流编排平台，集成了DeepSeek大模型，提供智能对话、Agent代码生成、工作流编排等核心功能。

### 🎯 核心定位

- **企业级AI Agent管理平台**
- **智能工作流编排系统**
- **DeepSeek大模型集成**
- **可视化Agent开发工具**

### 2. 🤖 AI Agent代码生成

**自然语言生成Agent代码**

- ✅ **智能代码生成** - 描述需求即可生成完整代码
- ✅ **多类型支持** - processor/analyzer/converter等
- ✅ **规范化输出** - 自动添加类型注解和文档
- ✅ **错误处理** - 自动包含异常捕获逻辑

**示例：**
```
用户输入："帮我生成一个数据处理Agent"

AI生成：
def data_processor(data: Dict[str, Any]) -> Dict[str, Any]:
    \"\"\"
    数据处理Agent
    
    Args:
        data: 待处理的数据字典
        
    Returns:
        处理后的数据
    \"\"\"
    try:
        # 处理逻辑
        result = {...}
        return {'success': True, 'result': result}
    except Exception as e:
        return {'success': False, 'error': str(e)}
```

## 🏗️ 技术架构

### 三层架构设计

```
┌─────────────────────────────────────────┐
│         Frontend（前端层）               │
│  ├─ templates/  (Jinja2模板)            │
│  ├─ static/css/ (样式)                  │
│  └─ static/js/  (交互逻辑)              │
├─────────────────────────────────────────┤
│         API Layer（API层）              │
│  └─ api/routes.py (REST API)            │
├─────────────────────────────────────────┤
│         Backend（后端层）                │
│  ├─ models.py    (数据模型)             │
│  ├─ database.py  (DAO层)                │
│  ├─ engine.py    (工作流引擎)           │
│  └─ llm_service.py (DeepSeek集成)       │
└─────────────────────────────────────────┘
```

### 技术栈

- **后端框架**: Flask 2.0+
- **数据库**: SQLite + SQLAlchemy
- **LLM集成**: DeepSeek API
- **前端**: HTML5 + CSS3 + JavaScript
- **数据可视化**: ECharts
- **加密**: Fernet (cryptography)

## 🚀 快速开始

### 1. 环境安装

```bash
# 创建conda虚拟环境
conda create -n agentforge python=3.9
conda activate agentforge

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置DeepSeek API Key

**方式1：首次访问时配置**
1. 访问 http://localhost:5000
2. 登录后点击「💬 DeepSeek 对话」
3. 点击右上角「配置」按钮
4. 输入您的API Key

**方式2：环境变量**
```bash
export DEEPSEEK_API_KEY="your-api-key-here"
```

**获取API Key：**
访问 https://platform.deepseek.com 注册并获取

### 3. 启动服务

```bash
cd "AI-agent"
python app.py
```

访问：http://localhost:5000

### 4. 登录系统

- **管理员账户**: `demo` / `demo123`
- **普通用户**: `user` / `user123`
